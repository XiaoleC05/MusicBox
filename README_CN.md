# MusicBox — 跨平台音乐聚合

> 一个搜索框，搜遍你所有音乐 App。无缝切歌，想听就听。

## 痛点在哪

不同音乐 App 的曲库各不一样。同一首歌，网易云有、QQ 音乐没有；换一首，又反过来。结果就是几个 App 来回切，歌单也东一个西一个，永远凑不齐。

**MusicBox** 把你的音乐账号串在一起。搜一首歌，它同时查你绑定的所有平台，自动挑能放的那个放。一个播放器，一份歌单，不用再切 App 了。

## 工作原理

MusicBox 不提供歌曲——它用你自己已有的会员：

1. 给每个平台填好登录信息（Cookie）
2. 你搜歌时，它同时去所有平台帮你找
3. 自动挑最合适的版本——音质好的优先、加载快的优先
4. 当前平台放不了这首歌？无声无息切到下一个平台继续放

**所有播放走你自己的账号和会员。** 这是个播放器，不是盗版工具。

## 功能一览

| 功能 | 能做什么 |
|------|---------|
| **跨平台搜歌** | 输歌名或歌手名，同时搜全部已绑平台 |
| **自动切源** | A 平台没版权？不提示、不报错，自动换 B 平台 |
| **完整播放器** | 播放、暂停、上下曲、拖进度、调音量，该有的都有 |
| **跨平台歌单** | 一份歌单里可以有不同平台的歌 |
| **优先高音质** | 有无损优先无损，没无损选最高码率 |
| **记住登录** | 配置一次，下次打开不用重来 |

## 技术选型

| 环境 | 后端 | 数据库 | 前端 | 特别说明 |
|------|------|--------|------|---------|
| 线上（个人用） | Go | PostgreSQL | React（Oxelia51） | 各平台适配插件 |
| 桌面（exe） | Go | SQLite | 内嵌 React | 同上 |

- 每个音乐平台一个独立适配器，想加新平台就写个新插件
- 线上版仅供我自己用，不开放给平台用户
- 桌面版所有人都能下载

## 怎么用

### 桌面版（推荐）

1. 去 [GitHub Releases](https://github.com/XiaoleC05/MusicBox/releases) 下载 `MusicBox.exe`
2. 双击运行
3. 在设置里贴上各平台 Cookie
4. 搜歌，听歌

### 线上版

[oxelia51.com](https://oxelia51.com) 上的在线版仅限我个人使用。请用桌面版。

## 关于版权
