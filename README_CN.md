# MusicBox — 跨平台音乐聚合播放器

> 一个搜索框，搜遍所有音乐平台。无缝切换，畅听无阻。

## 为什么需要 MusicBox？

不同音乐平台拥有不同的版权曲库。你想听的歌可能在网易云但不在 QQ 音乐上。结果你不得不在多个 App 之间来回切换，管理着分散的歌单，永远无法统一。

**MusicBox** 连接你已注册的音乐平台账号。一次搜索同时检索所有平台，自动选择有版权的音源播放。一个播放器，一个歌单，无需切换应用。

## 工作原理

MusicBox 不提供音乐——它使用你已有的平台账号：

1. 你为每个平台配置登录凭证（Cookie）
2. 搜索时，MusicBox 同时查询所有已连接平台
3. 自动选择最优音源——高音质优先、快响应优先
4. 当前平台无法播放时，静默切换到备用平台

**所有播放均使用你自己的账号和会员权益。** MusicBox 是播放器，不是盗版工具。

## 功能

| 功能 | 说明 |
|------|------|
| **跨平台搜索** | 一次搜索，同时检索所有已连接平台（歌曲名+歌手名） |
| **智能音源切换** | 平台A无版权？自动切换到平台B |
| **完整播放器** | 播放、暂停、切歌、进度条、音量控制 |
| **跨平台歌单** | 一个歌单可包含不同平台的歌曲 |
| **高音质优先** | 优先选择无损/高码率音源 |
| **持久登录** | 一次配置，下次自动登录 |

## MVP 范围

首发平台：**酷狗音乐**。验证可行性后再扩展其他平台。

## 技术栈

| 环境 | 后端 | 数据库 | 前端 | 特殊依赖 |
|------|------|--------|------|---------|
| 在线版（个人使用） | Go | PostgreSQL | React（Oxelia51） | 平台 HTTP 适配器 |
| 桌面版（exe，对外） | Go | SQLite | 内嵌 React | 同上 |

- **插件架构**：每个音乐平台一个独立适配器，方便扩展
- **在线版**：仅供开发者个人使用，不向平台用户开放
- **桌面版**：任何人都可下载使用

## 使用方式

### 桌面版（exe）— 推荐所有人使用

1. 从 [GitHub Releases](https://github.com/XiaoleC05/MusicBox/releases) 下载 `MusicBox.exe`
2. 运行即可
3. 在设置中配置音乐平台 Cookie
4. 开始搜索和播放

### 在线版 — 仅开发者

[oxelia51.com](https://oxelia51.com) 上的在线版仅供开发者个人使用，不对外开放。请使用 exe 版本。
